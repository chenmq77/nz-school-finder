## 4. UX 设计决策说明

| 设计决策 | 原因 |
|---------|------|
| A类用标签卡片（Tag Card） | 类型/性质/性别是离散分类值，卡片比表格更易扫读 |
| B类地址合并为单行 | 避免碎片化，符合用户阅读地址的习惯 |
| C类族裔用横向条形图 | 比例关系一目了然，比纯数字直观10倍 |
| C类国际生单独卡片 | 留学家庭第一关注点，需要视觉权重 |
| D类EQI用进度条+解读 | 家长不懂原始数值，需要翻译成人话 |
| D/E默认折叠 | 减少信息过载，80%用户看完ABC就够了 |
| 中英双语并排 | 目标用户可能是华人家长，降低理解成本 |
