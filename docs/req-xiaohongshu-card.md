# 小红书学校卡片导出 — 需求规格

## 概述
导出 1080x1440 (3:4) 两页图片，用于小红书系列内容"每天认识一所奥克兰高中"。

## 第1页（封面）

### 布局（从上到下）
1. 系列标签："每天认识一所奥克兰高中 No.{school_number}"
2. 校徽大图居中
3. 中文校名 + 英文校名居中
4. 标签组：公立/混校/9-13年级
5. NCEA L3 达标率（条件显示）
6. IB MYP / A-Level（条件显示，大字）
7. 关键信息列表
8. 品牌 "nz-school-finder"

### 条件显示规则
- NCEA L3：仅当 > 全国平均时显示，格式："NCEA L3 达标率 78.3% 高于全国 22.8 个百分点（全国 55.5%）"
- IB MYP：仅当 curriculum 包含 "IB" 时显示
- A-Level：仅当 curriculum 包含 "A-Level" 时显示

### 关键信息
- 学费 $XX,XXX/年
- 住宿 $XXX/周（有才写）
- 学生 X,XXX人 (国际生 X%)
- EQI XXX 社区条件优越/中等/薄弱

### 不包含
- 页码
- Emoji
- 学校特色描述（后续扩充）

## 第2页（NCEA 数据）
复用现有的 NCEA section 导出逻辑。

## 技术
- 尺寸：1080x1440px
- 背景：宣纸色 #F8F6F2
- 配色：PALETTE 千里江山图色系
- 导出：html2canvas
