# 小红书学校卡片导出 — 需求规格 v2

## 概述
为小红书系列"每天认识一所奥克兰本地高中"导出 1080x1440 (3:4) 封面图。

## 第1页（封面）布局

### 顶部色块（深蓝 PALETTE.blue[0]）
- "每天认识一所奥克兰本地高中" — 24px 白色
- "No.{school_number}" — 32px 白色粗体

### 校徽区域
- 160px 居中，白底圆角容器
- SVG 格式需先转 PNG（用 canvas 绘制）

### 校名
- 中文名 — 36px 黑色粗体
- 英文名 — 20px 灰色

### 标签组 — 20px
- 公立/私立 | 混校/男校/女校 | X-X年级

### NCEA L3 达标率（条件：高于全国平均才显示）
- "NCEA L3 达标率" — 20px 灰色
- "78.3%" — 56px 主色粗体
- "高于全国 ▲22.8 个百分点（全国 55.5%）" — 20px，▲22.8 用红色(#c04851)

### 课程体系色块（条件：有 IB/A-Level 才显示）
- "双轨课程（IB MYP）" — 24px 白字，青绿色块

### 关键信息 — 22px
- 学费 $XX,XXX/年
- 住宿 $XXX/周（有才写）
- 学生 X,XXX人（国际生 X%）
- EQI XXX 社区条件优越/中等/薄弱

### 底部品牌
- "nz-school-finder" — 20px 浅灰

## 字号规则
- 最小字号 = 20px（canvas 540px），确保手机可读
- 无低于 20px 的文字

## 技术要点
- SVG logo → canvas 转 PNG 后嵌入
- html2canvas 导出，scale:2，背景 #F8F6F2
- 颜色引用 PALETTE 对象
