## 关键设计变化（vs 当前 v2）
1. 标签（公立/混校/年级）用**醒目粗体大字**，不用色块背景
2. 标签散布在 logo 和校名的**四角**，不挤在一行
3. IB MYP 放在 NCEA 达标率**右侧**，不用色块，粗体即可
4. 底部 2x2：学费/住宿/国际生/社区条件
5. 配色用 PALETTE 千里江山图色系
6. 字号要大，粗体
