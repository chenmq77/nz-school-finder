# 导出功能 — 筛选结果导出为小红书图片

## 背景
用户筛选学校后，想把对比结果导出为图片分享到小红书等社交平台。

## 需求
在列表页增加"导出图片"按钮，将当前筛选结果 + 可见列导出为小红书竖版图片。

## 图片规格
- 尺寸: 1242 x 1660 px（3:4 竖版，适合小红书）
- 格式: PNG
- 内容: 标题区 + 筛选条件 + 数据表格 + 底部水印

## 图片模版布局
```
┌──────────────────────────────┐
│  NZ School Finder            │  ← 品牌标题
│  奥克兰公立中学对比           │  ← 自动生成的描述
│  筛选: Auckland / 公立 / Y11+ │  ← 当前筛选条件
├──────────────────────────────┤
│                              │
│  学校名     EQI  人数  学费   │  ← 当前可见列
│  ────────  ──── ──── ─────── │
│  奥克兰文法  386  2756 $25,750│
│  麦克林斯    385  2981 $25,000│
│  ...                        │
│                              │
├──────────────────────────────┤
│  nzschoolfinder.com          │  ← 水印
│  2026-03-26                  │  ← 日期
└──────────────────────────────┘
```

## 交互
- 按钮位置: "自定义列"按钮右边
- 点击后直接生成并下载 PNG（无弹窗选项，V1 简单版）
- 按钮文字跟随语言设置

## 技术约束
- 纯前端实现，使用 html2canvas 库
- 不依赖服务端渲染
- 导出内容 = 当前可见列 + 当前筛选后的全部数据
- 最多导出 20 所学校（超过则截断并提示）

## 导出表格样式
- 背景白色，文字深灰
- EQI 保留颜色标记（绿/橙/红）
- 表格行间距适中，字体 16-18px（适合手机阅读）
- 学校名只显示中文（有的话），无英文副行

## 不做（V1）
- 不做长图模式
- 不做 Excel/PDF 导出
- 不做导出选项弹窗
- 不做权限控制
